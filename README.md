# Smart Shop (Frontend Project)

A responsive e-commerce style frontend application built using pure HTML, CSS, and JavaScript.

This project was developed as part of a frontend university assignment, focusing on UI structure, responsiveness, and interactive behavior without using external frameworks.

---

## Overview

Smart Shop is a mock online store interface that demonstrates:

- Responsive layout (mobile-first approach)
- Dynamic product rendering
- Basic filtering and interaction logic
- Clean UI structure using semantic HTML
- Modular JavaScript functionality

The goal of this project was to simulate a modern shopping interface while keeping the implementation framework-free and easy to understand.

---

## Features

- Product listing grid
- Mobile-first responsive design
- Filtering functionality
- Interactive UI elements
- Clean layout and component structure
- Lightweight and dependency-free

---

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

No frameworks or external UI libraries were used.

## How to Run

### Option 1 – Open directly

Simply open:

```
index.html
```

in your browser.

### Option 2 – Run a local server (recommended)

From the project directory:

```bash
python -m http.server 8000
```

Then open:

```
http://127.0.0.1:8000
```

---

## Design Approach

The project follows a mobile-first philosophy:

- Layout optimized for small screens first
- Media queries used to scale up for larger screens
- Flexible grid system for product display
- Clear separation between structure (HTML), styling (CSS), and logic (JS)
